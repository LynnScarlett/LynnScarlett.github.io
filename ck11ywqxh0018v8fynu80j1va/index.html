箭头函数的使用

箭头函数和 function 命名的函数区别主要在于 this 的指向。function 函数的 this 值和执行环境有关，箭头函数的 this 值在定义时就确定。

map()/reduce()

具体可以看以下博客

map/reduce 廖雪峰的博客

map/reduce MDN Docs

简单来说 map()方法就是依次将数组内所有值执行一个过程（函数），返回值仍然是数组，比如每个元素加 1。

reduce()方法就是讲所有的元素执行一个过程后返回一个值比如求和。
